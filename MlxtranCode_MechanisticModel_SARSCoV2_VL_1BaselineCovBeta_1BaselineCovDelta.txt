DESCRIPTION: Model developed to fit viral load dynamics measured in NHPs after SARS-Cov-2 infection
			 With adjustment of viral infectivity and death rate of infected cells for two distinct baseline covariates (e.g. immunological biomarkers)

- The mechanistic model in derived from the Target cell-limited model 
  + Two distinct compartments Nasopharynx (N) & Trachea (T) jointly estimated
  + Addition of ODE compartment for inoculum (Vini) 
  + Use of volumes of trachea and nasopharynx in the definition of initial conditions.
- Observed viral Load (gRNA and sgRNA) in cp/ml ; Model in cp/ml

ODE compartments (X=N for nasopharynx and T for trachea):
 - TX: SARS-Cov-2 target cells
 - I1X: Infected cells 
 - I2X: Productively infected cells
 - VIX: Infectious virus
 - VNIX: Non-infectious virus
 - ViniX: Inoculated virus
 
 Model parameters:  
 - beta_N: viral infectivity in N
 - beta_N_pow: the power of 10 defining the viral infectivity (log10 transformation of beta)
 - beta_N_COV: viral infectivity in N adjusted for the first baseline covariate (immunological biomarker)
 - beta_T: viral infectivity in T
 - beta_T_COV: viral infectivity in T adjusted for the first baseline covariate (immunological biomarker)
 - fact_beta_T: parameter linking the estimation of beta in both N and T (no unit)
 
 - delta_N: death rate of infected cells in N (day-1)
 - delta_N_COV: death rate of infected cells in N adjusted for the second baseline covariate (immunological biomarker)
 - delta_T: death rate of infected cells in T (day-1)
  - delta_T_COV: death rate of infected cells in T adjusted for the second baseline covariate (immunological biomarker)
 - fact_delta_T: parameter linking the estimation of delta in both N and T (no unit)
 
 - P_N: viral production in N ()
 - P_T: viral production in T ()
 - fact_P_T: parameter linking the estimation of P in both N and T (no unit)
 
 - c: clearance of free denovo virions (day-1)
 - cI: clearance of inoculated virions (day-1)
 - k: eclispse phase (day-1)
 - g: parameter of transfert of virions between N and T (day-1)
 - mu: proportion of infectious virions (no unit)
 - alpha_VLSG: proportion rate between the log10-transformation of infected cells (I1+I2) and the subgenomic viral load (see observation model)

 - Weight: individual NHP weights (defined as regressor in the dataset and used to defined Trachea and Nasopharynx volumes with step functions) 
 - thresh_Weight: parameter of the step function defining the volumes of trachea and nasopharynx. 
 
 - beta_immuno1: regression coefficient evaluating the effect of the first baseline covariate on beta
 - beta_immuno2: regression coefficient evaluating the effect of the second baseline covariate on delta
 - Marker_baseline1: individual baseline value of the first covariate for which beta is adjusted (defined as regressor)
 - Marker_baseline2: individual baseline value of the second covariate for which delta is adjusted (defined as regressor)

Model outputs:
 - yTrachea_Total: gRNA viral load in Trachea
 - yTrachea_Sub: sgRNA viral load in Trachea
 - yNaso_Total: gRNA viral load in Nasopharynx
 - yNaso_Sub: sgRNA viral load in Nasopharynx


[LONGITUDINAL]
input = {beta_N_pow,fact_beta_T,fact_delta_T,delta_N,c,cI,k,g,mu,P_N,fact_P_T,alpha_VLSG,Weight,thresh_Weight,beta_immuno1,beta_immuno2,Marker_baseline1,Marker_baseline2}
Weight = {use=regressor}
Marker_baseline1 = {use=regressor}
Marker_baseline2 = {use=regressor}

EQUATION:
odeType = stiff

; Physiological quantities
; ------------------------
; Step function defining the volumes of the tracheal and nasopharyngeal compartments
if(Weight <= thresh_Weight)
	W_N = 4
	W_T = 2
else
	W_N = 5.5
	W_T = 3
end
; ------------------------

InoculumBaseline=4.37E9*5    ; Total amount of Inoculum injected in copies

; Initial Conditions (ODEs)
; ------------------------
t0=0

ViniT_0 = 0.9*InoculumBaseline/W_T   ; in cp/ml  - 90% of inoculum injected in Trachea
ViniN_0 = 0.1*InoculumBaseline/W_N	 ; in cp/ml  - 10% of inoculum injected in Nasopharynx
VIT_0 = 0   ; in cp/ml
VIN_0 = 0   ; in cp/ml
VNIT_0 = 0  ; in cp/ml
VNIN_0 = 0  ; in cp/ml

TN_0 = 1.25*10^5/W_N ; 	From Gonçalves et al. (2020) ; in cells/ml
TT_0 = 2.25*10^4/W_T ; 	From Gonçalves et al. (2020) ; in cells/ml
I1T_0 = 0   ; in cells/ml
I1N_0 = 0   ; in cells/ml
I2T_0 = 0   ; in cells/ml
I2N_0 = 0   ; in cells/ml
; ------------------------

; STRUCTURAL AND STATISTICAL MODELS 
; ---------------------------------
; + Statistical model (adjustment of beta and delta for two distinct baseline covariates)
; ----------
beta_N = 10^(beta_N_pow)
beta_N_COV = beta_N*exp(beta_immuno1*Marker_baseline1) ; The viral infectivty is adjusted for the first baseline covariate
beta_T_COV = beta_N_COV*exp(fact_beta_T)

delta_N_COV = delta_N*exp(beta_immuno2*Marker_baseline2) ; The death rate of infected cells is adjusted for the second baseline covariate
delta_T_COV = delta_N_COV*exp(fact_delta_T)	; 

P_T = P_N*exp(fact_P_T)
; ----------

; + Strustural model
; ----------
; Model for Trachea
ddt_TT    = -beta_T_COV*TT*VIT - mu*beta_T_COV*TT*ViniT ; 
ddt_I1T   = beta_T_COV*TT*VIT + mu*beta_T_COV*TT*ViniT - k*I1T ; 
ddt_I2T   = k*I1T - delta_T_COV*I2T ; 
ddt_VIT   = P_T*I2T*mu - c*VIT -beta_T_COV*TT*VIT -g*VIT +g*VIN ; 
ddt_VNIT  = P_T*I2T*(1-mu) - c*VNIT -g*VNIT +g*VNIN ;
ddt_ViniT = -cI*ViniT - mu*beta_T_COV*TT*ViniT ; 			Equation of the inoculum

; Model for Nasopharynx
ddt_TN    = -beta_N_COV*TN*VIN - mu*beta_N_COV*TN*ViniN; 
ddt_I1N   = beta_N_COV*TN*VIN + mu*beta_N_COV*TN*ViniN - k*I1N ; 
ddt_I2N   = k*I1N - delta_N_COV*I2N ; 
ddt_VIN   = P_N*I2N*mu - c*VIN -beta_N_COV*TN*VIN +g*VIT -g*VIN ;
ddt_VNIN  = P_N*I2N*(1-mu) - c*VNIN +g*VNIT -g*VNIN ; 
ddt_ViniN = -cI*ViniN - mu*beta_N_COV*TN*ViniN ;			Equation of the inoculum
; ----------
; ---------------------------------

; We impose positive values to I1 and I2
I1N_pos = max(I1N,0)
I2N_pos = max(I2N,0)
I1T_pos = max(I1T,0)
I2T_pos= max(I2T,0)

; OBSERVATION MODEL
; -----------------
; Observation Model 
if t == 0
  yNaso_Sub = -5  
  yNaso_Total = -5
  yTrachea_Sub = -5
  yTrachea_Total = -5
else 
  yNaso_Sub = alpha_VLSG*max(log10((I1N_pos+I2N_pos)),-5)    
  yNaso_Total = max(log10((VIN+VNIN+ViniN)),-5) 
  yTrachea_Sub = alpha_VLSG*max(log10((I1T_pos+I2T_pos)),-5)        
  yTrachea_Total = max(log10((VIT+VNIT+ViniT)),-5)   
end

OUTPUT:
output={yNaso_Sub,yNaso_Total,yTrachea_Sub,yTrachea_Total}
table = {TT,I1T,I2T,VIT,VNIT,ViniT,TN,I1N,I2N,VIN,VNIN,ViniN}
